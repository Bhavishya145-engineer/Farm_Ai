## **Troubleshooting:**

### **Backend won't start:**
```powershell
python -m pip install -r requirements.txt
```

### **Page shows "backend running?" error:**
- Make sure backend terminal shows "Application startup complete"
- Use http:// not file://
- Port 8000 must be free

### **Modal won't open:**
- F12 → Console tab for errors
- Try refreshing page (Ctrl+R)
- Clear browser cache

### **Login fails:**
- Check credentials are correct
- Verify backend is running
- Check F12 Network tab for errors

## **Browser DevTools (F12):**

### **Check localStorage:**
1. F12 → Application → Storage → localStorage
2. Should see `user` and `token` after login
3. Should be empty after logout

### **Check Network:**
1. F12 → Network tab
2. POST /login and /register appear
3. Status 200 = success
4. Status 400 = error

### **Check Console:**
1. F12 → Console tab
2. Should see no red errors
3. Warnings are OK
